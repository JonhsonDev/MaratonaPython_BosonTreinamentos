# 🐍 Intensivão de Python — Aprendendo e Compartilhando Conhecimento

Bem-vindo(a)! 👋

Este repositório é o registro da minha jornada de estudos em **Python**, unindo o conteúdo da **Faculdade de Engenharia de Software** com a **prática direta da programação**.
Aqui você vai encontrar códigos comentados, passo a passo, como se fossem **mini-aulas**, para que qualquer pessoa — iniciante ou curiosa — possa aprender junto comigo. 💻✨

---

## 🎯 Objetivo

Mais do que apenas guardar códigos, este projeto tem como propósito:

* Consolidar meus estudos de **lógica, sintaxe e conceitos fundamentais** de Python.
* Criar um material **didático e acessível** para quem também está começando.
* Mostrar como **engenharia de software e programação** se complementam na prática.

---

## 📚 O que você vai encontrar aqui

* 🧠 **Conceitos básicos**: variáveis, operadores, laços de repetição, funções, listas, etc.
* 🔍 **Códigos comentados** explicando o "porquê" de cada linha.
* 🧩 **Exemplos práticos** aplicados a problemas do dia a dia.
* 🚀 **Evolução contínua**: novos temas e arquivos sendo adicionados conforme meus estudos avançam.

---

## 💬 Como usar

1. Explore os diretórios e abra os arquivos `.py`.
2. Leia os comentários — eles foram feitos para **ensinar de forma leve e clara**.
3. Experimente executar os códigos no seu ambiente local (VS Code, Jupyter, Replit, etc).
4. Sinta-se à vontade para **clonar, estudar e contribuir!**

```bash
git clone https://github.com/JonhsonDev/IntensivaoPython.git
```
